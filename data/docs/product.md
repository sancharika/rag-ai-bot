Product Info:
Our platform is an AI-powered assistant built using Retrieval-Augmented Generation (RAG) and Vision AI to help users find information and understand images instantly. The system allows users to ask questions about any topic and receive accurate answers by retrieving relevant knowledge and generating human-like responses. In addition, the platform can analyze uploaded images and automatically generate captions and descriptive tags.

Key Features:
	•	AI Question Answering (RAG): Ask any question and get intelligent answers generated from retrieved knowledge sources.
	•	Image Captioning: Upload an image and the AI will generate a caption describing the image.
	•	Image Tag Generation: The AI also generates relevant tags for the uploaded image.
	•	Knowledge Retrieval System: Uses embeddings and vector search to fetch the most relevant information before generating answers.
	•	Multimodal AI Support: Supports both text-based queries and image-based inputs.
	•	Discord Bot Integration: Users can interact with the AI directly through Discord commands like /ask and /image.

Project Goal:
The goal of this project is to build a multimodal AI assistant that combines text-based knowledge retrieval (RAG) with image understanding (Vision AI) to create a smart assistant that can answer questions and describe visual content in real time.
